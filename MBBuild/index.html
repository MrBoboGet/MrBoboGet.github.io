MBBuild

MBBuild is a build system and command line application for compiling C/C++ code, but perhaps more importantly is a set of design principles for how build systems should be constructed.

MBBuild can also be seen both as a sibling and subproject of MBPacketManager, both aim at solving a related problem and are constructed for interoperability, and MBPacketManager is dependant on MBBuild, but MBBuild should also be usable as a separate command line utility.

MBBuild consist mainly of 3 parts, the build system file format, the command line utility, and the design philosophy behind it all.
